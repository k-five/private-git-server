Github

Github is a company that runs servers to manage software source code using git tool.

It has free public repositories and non-free private repositories.

With github we do this where the server is github.com:

// clone a repository
git clone git@github.com/k-five/private-git-server
// or
git remote add origin git@github.com/k-five/private-git-server


// for push 
git push -u origin master

// git remote -v
origin git@github.com/k-five/private-git-server.git (fetch)
origin git@github.com/k-five/private-git-server.git (push)

Gitlab

Gitlab is the same somehow, but has free private repositories.

It offers both public and private repositories for free.

With gitlab we do this where the server is gitlab.com

// clone a repository
git clone git@gitlab.com/k-five/private-git-server
// or
git remote add origin git@gitlab.com/k-five/private-git-server

// for push 
git push -u origin master

// git remote -v
origin git@gitlab.com/k-five/private-git-server.git (fetch)
origin git@gitlab.com/k-five/private-git-server.git (push)

Private Git Server

Private git server will be our own server that uses git tool as our version control system.

Although it is possible to have public repositories with a private-git-server, but the main purpose is to private repositories.

With our server we do this where the server is our own at private-git-server.ir

// clone a repository
git clone git@private-git-server.ir:private-git-server.git
// or
git remote add origin git@private-git-server.ir:private-git-server.git

// for push 
git push -u origin master

// git remote -v 
origin	git@private-git-server.ir:private-git-server.git (fetch)
origin	git@private-git-server.ir:private-git-server.git (push)
