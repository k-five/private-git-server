monitoring

Here is quick review of kinds of monitoring we can have.

Web based monitoring

Using HTML5 Web Socket API we can have a real-time monitoring using our web browsers.

Here is an example rtsm.ir.

Git based monitoring

Using git hooks we can have embedded monitoring as you can see in below screenshot.

Phone based monitoring

If we like to receive a kind of notification on our phone, we can setup Text Message Notification.

This is an asynchronous notification that unlike emailing we do not have to check our email periodically.

SSH based monitoring

Using SSH and keeping the connection alive help us to have a secure, fast, amazing monitoring.

In interesting part of using SSH is that we can redirect the out to our psuedo-terminal so we can have a single terminal for both pushing and monitoring.

Here is screenshot that after pushing to the server, the result can be redirected to us.
